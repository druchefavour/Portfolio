AirBnB: Tool Lending System


Back-To-Portfolio Frontend
Technologies & Skills:

html/html-5, css/css3, bootstrap, git/github, javascript,jQuery and ajax

In this project, an app is created which serves as a platform for people to share and borrow things from one another rather than buying these things and using them only a few times or renting them from professional rental companies at massive cost. One of the intents is to help reduce the number of things that are bought by consumers and therefore reduce our overall carbon footprint (i.e. potentially less waste in landfills). The app is an AirBnB-like storefront with functionalities that enable users to advertise owned items for share or hire. The impression and feel of the app should have a simplistic, current, socially-aware and user-friendly interface. The Frontend is written in HTML with Bootstrap CSS. Some parts of the HTML scripts are rendered using Express Handlebars. The Backend is powered by node server and the data are stored in MySQL Database.
